7 TRADES Style Guide

About

Our Service

Specialized in Chocolates

At 7 Trades all our staff specializes in making cakes and pastries. But most of all have a lot of passion in the art of chocolate. We place high value in the quality of our desserts.

Local + Organic

Our menu is largely based on what is in season. We try to use as many organic ingredients and local fruits as possible in our desserts.

Adventurous

Our menu is constantly changing, we always try new combinations of different / new ingredients and see how they combine and chocolate.

About

The main navigation bar will be fixed at the side of the page even when scrolling. The background will be white with light blue links that turn dark blue when hovered upon.

This navigation bar will be the main way for a user to access different pages on the website. Used for medium to large sized screens.

Tabs

Inital State

Css Selector

.tabs

.tab_links

Code

< div class="tabs" >
< button class="tab_links" onClick="openTab(event, Group1)"> Tab 1 </button>
< button class="tab_links" onClick="openTab(event, Group2)"> Tab 2 </button>
< button class="tab_links" onClick="openTab(event, Group3)"> Tab 3 </button>
</div>

About

Tabs are used for ocassions when there are too much content on one page and needs to be categorized under a various groups

Hover State

Css Selector

.tab_links: active, .tab_links:hover, .tab_links:focus

Code

<button class="tab_links onClick="openTab(event, Group2)"> Tab 2 </button>

About

When hovered upon the text will change to the lighter blue colour, and when selected the light blue line will go under the tab selected.
